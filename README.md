# Ferro en Acción

Este repositorio reúne documentos públicos de Ferrocarril Oeste, ordenados para que sea más fácil encontrarlos y entender su continuidad.

## Qué vas a encontrar

- Actas de Comisión Directiva.
- Actas de Asamblea.
- Balances asociados.
- Convocatorias y otros documentos de apoyo.

## Cómo está ordenado

- Cada carpeta agrupa documentos por tipo.
- Los nombres de archivo usan fecha y una referencia corta.
- Los `index.md` resumen qué hay en cada carpeta y señalan faltantes o particularidades.

## Para qué sirve

- Consultar documentos históricos de forma ordenada.
- Seguir la secuencia de actas y balances.
- Encontrar rápido la fecha y el tipo de cada archivo.

## Nota

Algunos documentos están en PDF, otros en DOCX o en imágenes, porque así aparecieron publicados originalmente.
